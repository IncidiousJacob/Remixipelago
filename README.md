Setup Guide

Requires bizhawk 2.10, version 2.0.1 of the smash remix mod, and a legally obtained copy of Smash Bros 64 (Usa) to play

1. Use the Smash remix website to patch your rom as yyou normally would for Smash Remix
2. Download the apworld from the releases tab, and place it in your custom_worlds folder
3. Generate a yaml through a template yaml or through the options creator located in you ap launcher
4. generate the multiworld
5. obtain you .apsr patch from either the host website, or the output folder produced when generating a multiworld
6. Open the patch through the ap launcher
7. When prompted select your Patched Smash remix rom, and emuhawk executable
8. Have fun
